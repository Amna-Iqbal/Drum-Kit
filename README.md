# 🥁 Drum Kit

A fun and interactive **Drum Kit** project built with HTML, CSS, and JavaScript. Press keys or click buttons to play different drum sounds with visual feedback.

🔗 **Live Demo**: [amna-iqbal.github.io/Drum-Kit](https://amna-iqbal.github.io/Drum-Kit)

---

## 🎯 Features

- ⌨️ **Keyboard Events**  
  Press specific keys (`w`, `a`, `s`, `d`, `j`, `k`, `l`) to play drum sounds.

- 🖱️ **Mouse Click Support**  
  Click on the on-screen drum buttons for the same effect.

- 🔊 **Audio Playback**  
  Each key/button is linked to a unique drum sound (e.g., snare, tom, crash, kick).

- ✨ **Button Animation**  
  Visual feedback through a brief animation whenever a key is pressed or a button is clicked.

---

## 🛠️ Technologies Used

- **HTML5**  
- **CSS3**  
- **JavaScript (Vanilla)**

---

## 📂 How to Use

1. Visit the live demo:  
   [https://amna-iqbal.github.io/Drum-Kit](https://amna-iqbal.github.io/Drum-Kit)

2. Press any of the following keys on your keyboard:  
   `w`, `a`, `s`, `d`, `j`, `k`, `l`  
   OR  
   Click the buttons on the screen to play corresponding sounds.
